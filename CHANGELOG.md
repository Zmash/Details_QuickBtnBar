# Changelog

## 1.0.0
Initial public release.

- Quick-switch button bar for Details!: left-click switches the display, right-click toggles Current/Overall
- Free mode (movable bar) and docked mode (one bar per Details! window)
- Per-window broker assignment and visibility scopes (always / dungeon / raid / both)
- Drag & drop button reordering
- Accent color (defaults to class color), background opacity, vertical offset
- Localization: English and German, follows client language by default, selectable in options
- Minimap button (can be disabled in options)
